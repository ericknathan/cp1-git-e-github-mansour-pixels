## 📖 Descrição do Projeto

A **Social Connect**  é uma plataforma inovadora com a missão de conectar você às pessoas certas. O projeto visa construir um ambiente digital dinâmico e interativo, focado na criação de comunidades e no fortalecimento de redes de contatos. Para acessar o projeto, basta utilizar o link a seguir: https://ericknathan.github.io/cp1-git-e-github-mansour-pixels/

## 🚀 Como Executar o Projeto

### Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina a ferramenta [Git](https://git-scm.com). Além disto, é bom ter um editor para trabalhar com o código, como o [VSCode](https://code.visualstudio.com/).

### Passo a Passo

```bash
# 1. Clone este repositório
$ git clone https://github.com/ericknathan/cp1-git-e-github-mansour-pixels.git

# 2. Acesse a pasta do projeto no terminal
$ cd cp1-git-e-github-mansour-pixels

# 3. Abra o projeto no seu editor de código (se usar o VSCode)
$ code .
```
